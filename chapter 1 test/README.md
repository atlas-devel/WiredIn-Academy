
# Coin Toss Game

A simple Python learning project from **WiredIn Academy** in collaboration with **TechGym (Japan)**.

## Overview

This is a beginner-friendly coin toss game where you bet on heads or tails, and the computer randomly selects a side. Win by matching the coin result!

## How to Play

1. Run the script
2. Input your bet:
    - `0` for heads
    - `1` for tails
3. The computer flips a coin
4. See if you win or lose

## Requirements

- Python 3.x
- No external dependencies

## Running the Game

```bash
python coin_toss.py
```

## Features

- Input validation (accepts only 0 or 1)
- Error handling for non-numeric input
- Clear game feedback
- Simple, clean code structure

## Learning Objectives

- User input handling
- Conditional logic
- Random number generation
- Functions and return types
- String formatting
